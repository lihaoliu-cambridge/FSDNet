## Requirements

* PyTorch1.3.0
* Python3.6
* Cupy [Installation Guide](https://docs-cupy.chainer.org/en/stable/install.html#install-cupy)

  
## Train and Test

1. Clone this repository:          
   ```git clone https://github.com/xw-hu/FSDNet.git
   ```
2. Train:
   ```python3 train.py    
   ```
3. Test:
    ```python3 infer.py
    ```
